
# ModemManager-1.0

GJS TypeScript type definitions for ModemManager-1.0, generated from library version 1.21.3 using [ts-for-gir](https://github.com/gjsify/ts-for-gjs) v3.0.0-beta.12.

ModemManager provides a unified high level API for communicating with (mobile broadband) modems.

## Install

To use this type definitions, install them with NPM like this:
```bash
npm install @girs/modemmanager-1.0
```

